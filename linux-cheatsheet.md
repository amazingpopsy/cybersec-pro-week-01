# 10 Linux Commands
1. **mkdir** - It is used to create a directory e.g mkdir cybersecurity
2. **rm**  - It is used to remove a file or directory e.g rm file.txt
3. **cp** - It is used to copy a file from one directry to another e.g cp desktop/file.txt /downloads
4. **mv** - It is used to move a file from one directory to another e.g mv desktop/file.txt /downloads
5. **ls** - It is used to fo list the content of a direcstory e.g ls /desktop
6. **top** - It is used to show processes running on the system e.g top
7. **cd** - It is used to change from one directory to another e.g cd /downloads
8. **cat** - It is used to display the content of a file on the terminal e.g cat file.txt
9. **wc** - It is used to count the number of words in a file e.g wc file.txt
10. **sudo** - It is used perform a function as a super or admin user e.g sudo apt update
